# 小区物业管理系统 java819

## 项目概述

小区物业管理系统是基于Java EE技术栈开发的一个Spring Boot项目。该项目集成了前端Vue框架，后端使用MyBatis-Plus进行数据库操作，MySQL作为数据存储。

## 技术栈

- 后端开发框架：Spring Boot
- 前端开发框架：Vue
- 数据库操作框架：MyBatis-Plus
- 数据库：MySQL
- 运行环境：JDK 1.8、IDEA、Maven 3.6.1（其他版本可自测）

## 功能模块

- 用户模块：实现住户个人信息管理、房屋报修和投诉功能。
- 管理员模块：实现对小区信息的操作管理及处理住户需求。

## 系统角色

- 住户：可以登记个人信息，提交报修和投诉。
- 管理员：管理小区信息，响应用户需求。

## 运行环境

- JDK 1.8
- IDEA
- MySQL 8.0
- Maven 3.6.1
- Navicat（推荐，其他版本可自测）

## 部署步骤

1. 环境搭建：安装JDK、IDEA、MySQL、Maven等。
2. 数据库配置：使用MySQL数据库，执行提供的SQL脚本初始化数据。
3. 项目导入：通过IDEA或Maven导入项目。
4. 依赖管理：使用Maven管理项目依赖。
5. 配置修改：根据实际环境修改配置文件。
6. 项目运行：执行Spring Boot应用程序。
